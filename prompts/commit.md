Help me commit the changes made so far.

**Always use conventional commit prefixes:**
- `feat:` - New features or enhancements
- `fix:` - Bug fixes
- `docs:` - Documentation updates
- `refactor:` - Code refactoring without functional changes
- `perf:` - Performance improvements
- `test:` - Adding or updating tests
- `chore:` - Maintenance tasks

**Examples:**
```bash
git commit -m "feat: add parameterized request count to test traffic generator"
git commit -m "docs: update monitoring references from CloudWatch to DynamoDB"
git commit -m "perf: implement boto3 session reuse for 96.7% performance improvement"
```
